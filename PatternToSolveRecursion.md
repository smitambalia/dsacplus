### Include and Exclude pattern 

-- Substring and Subsequence 

 - Substring : string should not break the sequence 

        Example : "Smit"
                - "mit"
                - "it"
                - "i"
                - "t"

- Subsequence string :  string can break the sequence however not the order 
        total = 2 ^4 
        Example : "Smit"
                - "S"
                - "m"
                - "i"
                - "t"
                - "Sm"
                - "Si"
                - "St"
                - "Smi"
                - "mi"
                - "mt"
                - "mit"
                - "i"
                - "it"
                - "t"
                - " "
